## Silverfish      =>    蠹鱼，衣鱼(蛀食织物、纸张等的小虫)

 取自陆游的诗,去做一个食遍文海的蠹鱼
 ```
	
    吾生如蠹鱼，亦复类熠耀。
       
    一生守断简，微火寒自照。
    
    区区心所乐，那顾世间笑。
    
    闭门谢俗子，与汝不同调
```
